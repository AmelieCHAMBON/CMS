Les plug-ins

Qu'est-ce qu'un plug-in ?
" Programme qui interagit avec un logiciel principal pour lui apporter de nouvelles fonctionnalités. " - AT Internet

Les plug-in spécialisés pour le e-commerce peuvent vous proposer plusieurs types de fonctionnalités.
Tout d'abord, le site peut vous inviter à modifier le thème du cms, dans le but d'optimiser la page.
De même, il existe des fonctionnaités de gestion des stocks, gestion des transactions, des livraisons etc...

Regardons de plus près deux plug-in de WordPress en adéquation avec les sites de e-commerce...

WooCommerce :
Tout d'abord, WooCommerce qui est extension open source pour créer une boutique en ligne, qu'importe la taille de l'entreprise.

Fonctionnalités :
  • Catégories de produits
  • Les produits et ses variantes
  • Inventaire automatique
  • Différents processeurs de paiement pour les différents pays
  • Différents transporteurs
  • Les informations légales de la boutique


WPshop :
WPshop est une extension gratuite et française proposant des services complets, simple et performants.

Fonctionnalités :

  • Modules de paiements CB
  • Module de livraison via So-Colissimo
  • Module de gestion
  • Thèmes WordPress optimisés pour le e-commerce
